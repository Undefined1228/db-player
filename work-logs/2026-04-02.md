## 뷰 생성/수정/삭제 기능 구현

- 변경 파일: `src/main/db/metadata.ts`, `src/main/index.ts`, `src/preload/index.ts`, `src/preload/index.d.ts`, `src/renderer/src/components/CreateViewDialog.svelte`, `src/renderer/src/components/SchemaObjectContextMenu.svelte`, `src/renderer/src/components/SchemaTree.svelte`, `src/renderer/src/components/Sidebar.svelte`
- 내용:
  - `metadata.ts`: `createView` (CREATE OR REPLACE VIEW), `alterView` (쿼리 변경 + RENAME 트랜잭션), `dropView` (DROP VIEW) 함수 추가

  - `SchemaObjectContextMenu.svelte`: view 타입에 "뷰 수정" 메뉴 항목 추가
  - `SchemaTree.svelte`: Views 헤더 우클릭 컨텍스트 메뉴 ("뷰 생성") 추가, `onCreateView` / `onAlterView` / `onDropView` prop 추가
  - `Sidebar.svelte`: CreateViewDialog 및 뷰 삭제 AlertDialog 상태 관리 및 렌더링

## 인덱스 관리 기능 구현 (PostgreSQL)

- 변경 파일: `src/main/db/metadata.ts`, `src/main/index.ts`, `src/preload/index.ts`, `src/preload/index.d.ts`, `src/renderer/src/components/CreateIndexDialog.svelte`, `src/renderer/src/components/SchemaTree.svelte`, `src/renderer/src/components/Sidebar.svelte`
- 내용:
  - `metadata.ts`: `IndexInfo` 인터페이스 추가 (`name`, `unique`, `columns[]`). `TableInfo.indexes` / `MatViewInfo.indexes`를 `string[]`에서 `IndexInfo[]`로 변경. 인덱스 쿼리를 `pg_index`와 조인해 unique 여부·컬럼 목록 포함하도록 개선. `createIndex` (CREATE [UNIQUE] INDEX), `dropIndex` (DROP INDEX) 함수 추가
  - `index.ts`: `index:create`, `index:drop` IPC 핸들러 추가
  - Preload: `createIndex`, `dropIndex` API 바인딩 및 타입 정의
  - `CreateIndexDialog.svelte`: 인덱스 이름·유니크·메서드(btree 등)·컬럼 선택(순서 변경, ASC/DESC) UI 구현
  - `SchemaTree.svelte`: Indexes 헤더 우클릭 → "인덱스 생성", 개별 인덱스 항목 우클릭 → "인덱스 삭제". 인덱스 아이템에 UNIQUE 배지 및 컬럼 목록 표시
  - `Sidebar.svelte`: CreateIndexDialog 및 인덱스 삭제 AlertDialog 상태 관리
- 커밋: ⏳ 미커밋

## 세션 모니터 탭 기능 구현
