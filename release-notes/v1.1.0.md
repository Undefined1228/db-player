## 새 기능

### 뷰(View) 생성 · 수정 · 삭제 (PostgreSQL)

PostgreSQL 스키마에서 뷰를 GUI로 직접 관리할 수 있습니다.

**생성**
- 사이드바에서 Views 헤더 우클릭 → **뷰 생성**
- 뷰 이름과 SELECT 쿼리를 입력하면 `CREATE OR REPLACE VIEW`로 생성

**수정**
- 뷰 항목 우클릭 → **뷰 수정**
- 기존 SELECT 쿼리가 자동으로 로드되며, 쿼리 변경 및 이름 변경 지원
- 쿼리 변경과 이름 변경이 하나의 트랜잭션으로 처리

**삭제**
- 뷰 항목 우클릭 → **뷰 삭제**
- 삭제 확인 다이얼로그 후 `DROP VIEW` 실행

---

### 인덱스 관리 (PostgreSQL)

PostgreSQL 테이블의 인덱스를 GUI로 생성·삭제할 수 있습니다.

- 스키마 트리에서 테이블 확장 → Indexes 헤더 우클릭 → **인덱스 생성**
- 인덱스 이름, UNIQUE 여부, 인덱스 방식(B-tree / Hash / GIN / GiST / BRIN), 컬럼 및 정렬 방향(ASC/DESC) 설정
- 인덱스 항목 우클릭 → **인덱스 삭제**
- 인덱스 항목에 UNIQUE 배지 및 컬럼 목록 표시

---

### 세션 모니터 탭

연결에 대한 실시간 DB 상태를 GUI로 확인할 수 있습니다.
사이드바에서 연결 우클릭 → **세션 모니터**로 진입합니다.

**활성 세션**
- 현재 DB에 연결된 세션 목록 조회 (자기 자신 제외)
- 세션 상태(active / idle / idle in transaction 등)를 색상 뱃지로 구분
- 실행 시간, 대기 이벤트, 실행 중 쿼리 표시
- 세션 강제 종료: 쿼리 취소(`state = active`일 때만) / 연결 종료 — 확인 다이얼로그 필수

**잠금 현황**
- Lock Wait 상태인 세션과 이를 차단하는 세션을 쌍으로 표시
- 잠금 유형, 대상 테이블, 각 세션의 실행 쿼리 표시

**테이블 통계**
- 전체 크기 / 테이블 크기 / 인덱스 크기 / 추정 행 수
- PostgreSQL 전용: Dead Tuples 수, 마지막 VACUUM / Auto-VACUUM 시각

**자동 새로고침**
- 5초 / 10초 / 30초 / 60초 주기 설정 가능
- 탭 전환 시 해당 섹션 데이터를 자동으로 fetch

지원 DB: PostgreSQL, MySQL, MariaDB

## 개선 사항

### 뷰 에디터 SQL 자동완성

뷰 생성/수정 다이얼로그의 쿼리 입력란이 SQL 에디터와 동일한 수준으로 업그레이드되었습니다.

- 스키마·테이블·컬럼 자동완성 (현재 연결 기준)
- PostgreSQL 구문 하이라이팅
- 라이트/다크 테마 자동 전환
- Undo/Redo, Tab 들여쓰기 지원

### 테이블 트리 하위 항목 그룹화

테이블 확장 시 하위 항목이 세 그룹으로 정리됩니다.

- **Columns** — 컬럼 이름 및 타입 목록
- **Foreign Keys** — 제약조건 이름 + `로컬컬럼 → 참조테이블(참조컬럼)` 형식
- **Indexes** — 인덱스 이름, UNIQUE 배지, 컬럼 목록

각 그룹은 독립적으로 펼침/접기 가능합니다.
