## 새 기능

### 데이터 뷰어 컬럼 타입별 입력 방식 개선

**boolean**
- 체크박스 직접 수정 방식에서 더블클릭 후 `true` / `false` / `NULL` 드롭다운 선택 방식으로 변경

**date / timestamp / time**
- 달력·시간 피커 입력 (`input type=date` / `datetime-local` / `time`)

**숫자**
- 숫자가 아닌 값 입력 시 빨간 테두리 경고 표시

**복붙 단축키 (macOS)**
- 편집 메뉴(cut / copy / paste / selectAll) 추가로 macOS 복붙 단축키 활성화
