## 새 기능

### MySQL / MariaDB 전체 기능 지원

MySQL 및 MariaDB 연결에서 PostgreSQL과 동등한 수준의 스키마 탐색 및 데이터 관리 기능을 사용할 수 있습니다.

**스키마 탐색**
- 사이드바에서 테이블 · 뷰 · 함수 · 프로시저 목록 조회
- 테이블 확장 시 Columns / Foreign Keys / Indexes 그룹으로 구분 표시
- 스키마 오브젝트 우클릭 컨텍스트 메뉴 지원

**DDL 조회**
- 테이블 · 뷰 우클릭 → **DDL 보기**
- `SHOW CREATE TABLE / VIEW` 결과를 포맷팅하여 표시

**데이터 뷰어**
- 테이블 더블클릭 또는 우클릭 → **데이터 보기**
- 행 수 표시, 페이지네이션 지원

**테이블 생성 · 수정**
- 사이드바 우클릭 → **테이블 생성**: 컬럼 정의, PRIMARY KEY, NOT NULL, DEFAULT, AUTO_INCREMENT, 문자셋/콜레이션 설정
- 테이블 우클릭 → **테이블 수정**: 기존 컬럼 수정, 컬럼 추가/삭제, 테이블명 변경

---

### MariaDB EXPLAIN 전용 파서

MariaDB의 `EXPLAIN FORMAT=JSON` 출력 구조를 정확하게 파싱합니다.

- MariaDB는 MySQL 8.0과 달리 `cost`/`rows` 필드를 직접 노출 (MySQL의 `cost_info.query_cost` / `rows_examined_per_scan` 구조와 상이)
- `normalizeMariadbTable` / `normalizeMariadbBlock` 전용 파서 적용
- 실행 계획 트리 뷰어에서 MariaDB 쿼리 정상 표시
